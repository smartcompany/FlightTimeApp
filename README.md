# Flight Time Games

비행기 내에서 인터넷 연결 없이 실행 가능한 Flutter 미니게임 10개 모음입니다.

## 게임 목록

1. **뱀 게임** - 클래식한 뱀 게임
2. **틱택토** - 3x3 보드에서 하는 틱택토
3. **기억력 게임** - 카드 매칭 게임
4. **숫자 퍼즐** - 15-퍼즐 게임
5. **가위바위보** - 컴퓨터와 대전
6. **행맨** - 단어 맞추기 게임
7. **퀴즈** - 퀴즈 게임
8. **반응속도** - 반응 속도 테스트
9. **색깔 맞추기** - 색상과 이름 매칭 게임
10. **단어 찾기** - 단어 찾기 퍼즐

## 설치 및 실행

1. Flutter SDK가 설치되어 있어야 합니다.
2. 프로젝트 디렉토리로 이동:
   ```bash
   cd client
   ```
3. 의존성 설치:
   ```bash
   flutter pub get
   ```
4. 앱 실행:
   ```bash
   flutter run
   ```

## 특징

- 완전 오프라인 작동
- 인터넷 연결 불필요
- 모든 게임이 독립적으로 작동
- 간단하고 직관적인 UI
